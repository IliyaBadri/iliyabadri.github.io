EnterPass Open-Source Password Manager

Securely store and manage your passwords with AES encryption and Argon2 master key hashing.

EnterPass is an open-source password manager developed by EnterACE repository. It is designed to securely store and manage passwords using AES encryption and Argon2 key hashing.


Features

- AES encryption in CBC mode for secure storage of passwords

- Argon2 key hashing for enhanced password security

- User-friendly graphical user interface using tkinter library

- Ability to add, edit, and delete passwords in the password manager

- Encrypted storage of passwords in a CSV database file

Installation

You can either use the source code and run it as an python script or install it via the windows installer.



First you need to download either the program source or the installer.


Get started with EnterPass password manager today:

Run it via python source code (recomended) Download the windows installer

After the download was complete:


If you have downloaded the source code you can follow the installation proccess here

If you have downloaded the installer:

1. Open the program as adminstraitor

2. Enter your prefered master password (32 chacacters is recommended. No more. No less.)

3. Hit continue.

4. Enter your master password again.

5. Thats it!

Usage

Note: If you have downloaded the windows installation program you will need to run the program as adminstraitor to grant the app to write changes to the database file.

Each time you open the program you will be asked for the master password.

Note:

If you lose your master password there is NO WAY to retrive it back.

After the first opening of the program you will be asked for the master password and the password you enter will not be directly saved to the file system and it will be hashed in Argon2 hashing algorithm.

This way the program can check your master pasword with the hash but the program itself does not store the password in plain text.

The only way you can retrive your password back is cracking the Argon2 hash which will be stored in key.hash file next to the python script or windows executeable.

And please note that Argon2 takes an ethernity to crack.

After the download was complete:


When you have entered your master password the password manager will open and decrypt the passwords database.

In this panel to add a password you will need to enter your prefered fields in the Inputs frame on the right side of the screen and then click on the "Add new password" button.

To edit a field in the database you need to first enter the ID of the password row and then overwrite that with the other input fields and then click "Edit password by ID" button.

To delete a password you will need to enter the password row ID in the input panel and click on the "Delete password by ID" button.
